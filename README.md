# Travel Itinerary Editor

A web-based application to create, edit, and manage travel itineraries.

## Features

- Create and manage multi-day travel itineraries
- Add activities with time and descriptions
- Support for nested sub-items
- Custom sections for instructions and notes
- Auto-save functionality
- Dark/Light theme support
- Export to formatted text (WhatsApp-ready)
- Data persistence using local storage

## How to Use

### Basic Information Section

1. **Title**: Enter the main title of your itinerary
2. **Dates**: Specify the date range for your trip
3. **Organizer**: Enter the organizer's name
4. **Contact Numbers**: Enter contact numbers separated by commas (e.g., 9500008901, 9500025901)

### Managing Days

1. **Add Day**: Click the green "+ Add Day" button to create a new day
2. **Edit Day Title**: Click on the day title field to edit (e.g., "Day 1: Saturday")
3. **Remove Day**: Click the "Remove Day" button to delete an entire day
4. **Collapse/Expand**: Use the тЦ╝ arrow to collapse or expand day sections

### Managing Activities

1. **Add Activity**: Click "+ Add Activity" within a day section
2. **Edit Time**: Enter the time in the time field (e.g., "01:45 AM")
3. **Edit Description**: Enter the activity description in the text area
4. **Add Sub-items**: Click "+ Add Sub-items" to create a nested list under an activity
5. **Delete Activity**: Click the ├Ч button next to the time field

### Managing Sections

1. **Add Section**: Click the green "+ Add Section" button
2. **Edit Section Title**: Enter a title for your section (e.g., "Important Instructions")
3. **Add Items**: Click "+ Add Item" to add entries
4. **Edit Items**: Each item has a title and description field
5. **Delete Items**: Click the ├Ч button to remove an item
6. **Remove Section**: Click "Remove Section" to delete the entire section

### Footer Message

1. Expand the "Footer Message" section
2. Edit the footer text that will appear at the end of your itinerary

### Menu Options (тШ░)

Click the hamburger menu in the top-right corner to access:

1. **Toggle Theme**: Switch between light and dark modes
   - Moon icon = Light mode (click to switch to dark)
   - Sun icon = Dark mode (click to switch to light)

2. **Reset to Original**: Restore the default Shirdi Sai Yatra itinerary
   - **Warning**: This will erase all your changes permanently

### Saving and Exporting

1. **Auto-Save**: Data is automatically saved as you type (after 500ms of inactivity)
2. **Manual Save**: Click the "ЁЯТ╛ Save" button to save immediately
3. **Copy Formatted Text**: Click "ЁЯУЛ Copy Formatted Text" to copy the itinerary in WhatsApp-ready format
   - The text uses `*` for bold formatting
   - Properly formatted with separators
   - Ready to paste into WhatsApp, email, or any text application

### Data Storage

- All data is stored in your browser's local storage under the key `sai_itinerary_data`
- Data persists even after closing the browser
- To completely clear data, use "Reset to Original" from the menu
